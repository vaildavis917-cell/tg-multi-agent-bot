# 🤖 Мульти-агентный Telegram-бот (Claude Opus 4.6)

Это приватный Telegram-бот, построенный на базе передовой языковой модели Claude Opus 4.6 (через OpenRouter). Бот представляет собой мощный инструмент для инвесторов, трейдеров и аналитиков, объединяющий в себе 8 специализированных ИИ-агентов с уникальными ролями и экспертизой.

## 🎯 Что даёт проект пользователю?

Бот превращает Telegram в карманный аналитический центр инвестиционного банка. Вместо того чтобы писать сложные промпты каждый раз, пользователь просто выбирает нужного эксперта (агента) из меню, предоставляет базовые вводные данные (например, тикер акции или состав портфеля), и получает профессиональный, структурированный разбор.

Это экономит часы времени на сбор информации, анализ графиков, чтение отчётностей и расчёт финансовых моделей.

## 🌟 Основные возможности и функции

### 1. Система Мульти-Агентов
В бот встроено 8 предустановленных агентов, каждый из которых имеет свой уникальный "характер" и системный промпт, настроенный на выполнение конкретных финансовых задач:

1. **📊 Скринер акций Goldman Sachs** — отбирает топ-10 акций под конкретные инвестиционные цели, анализирует P/E, рост выручки, дивиденды и оценивает риски.
2. **🏦 DCF-оценка Morgan Stanley** — строит модель дисконтированных денежных потоков, прогнозирует выручку, рассчитывает WACC и выдает справедливую стоимость акции.
3. **🛡 Оценка рисков Bridgewater** — проводит стресс-тесты портфеля, анализирует корреляцию активов, географические риски и предлагает стратегии хеджирования.
4. **📈 Анализ отчётности JPMorgan** — готовит превью перед публикацией квартальных отчётов, анализирует консенсус-прогнозы и историческую реакцию рынка.
5. **💼 Конструктор портфеля BlackRock** — создает индивидуальный инвестиционный портфель с распределением активов, рекомендациями ETF и графиком ребалансировки.
6. **📉 Технический анализ Citadel** — разбирает тренды, уровни поддержки/сопротивления, скользящие средние, RSI, MACD и графические паттерны для поиска точек входа.
7. **🎓 Дивидендная стратегия Гарварда** — формирует портфель для пассивного дохода, оценивает надежность выплат (payout ratio) и эффект сложного процента (DRIP).
8. **🔍 Конкурентный анализ Bain** — проводит SWOT-анализ сектора, сравнивает маржинальность конкурентов и определяет лучшую акцию для покупки.

### 2. Режим "Свободный чат"
Помимо общения со специализированными агентами, пользователь может переключиться в режим свободного общения. В этом режиме бот работает как обычный ChatGPT/Claude — без жестких рамок системного промпта, что идеально подходит для повседневных вопросов, написания текстов или перевода.

### 3. Контекстная память (История диалогов)
Бот запоминает ход беседы. Можно попросить агента "уточни этот момент" или "пересчитай с другими данными", и он поймет контекст. В любой момент контекст можно сбросить специальной кнопкой, чтобы начать новую задачу с чистого листа.

### 4. Приватность и Whitelist (Белый список)
Бот является закрытым. Никто не сможет им воспользоваться, пока администратор не добавит Telegram ID пользователя в Whitelist.
* **Новая фича:** При добавлении пользователя теперь можно указывать тег (имя или заметку), чтобы в админ-панели было понятно, кому принадлежит ID (например: `7415656277 Влад`).

### 5. Встроенная Админ-панель
Прямо внутри Telegram, без необходимости заходить на сервер, администратор может:
* **Управлять доступом:** Добавлять и удалять пользователей, просматривать список разрешенных лиц.
* **Управлять агентами:** Создавать новых агентов, редактировать их имена, эмодзи, описания и, самое главное, их системные промпты. Любого агента можно временно отключить.
* **Смотреть статистику:** Отслеживать количество потраченных токенов (входящих и исходящих) для контроля расходов по API OpenRouter.

## 🛠 Техническая реализация
* **Язык:** Python 3.10+
* **Фреймворк:** aiogram 3 (полностью асинхронный, быстрый, с поддержкой современных Inline-клавиатур).
* **База данных:** SQLite (локальная, быстрая, не требует отдельного сервера).
* **Интеграция LLM:** OpenRouter API (модель `anthropic/claude-3-opus:beta` или эквивалентная 4.6).
* **Деплой:** VPS на базе Ubuntu, автоматический запуск через `systemd`, что гарантирует автоподнятие бота при перезагрузке сервера.
